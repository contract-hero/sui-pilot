Claude Code Plugin

Turn Claude into a Sui/Move expert — grounded in current docs.

sui-pilot bundles 747 documentation files from six MystenLabs corpora, bridges the Move LSP and Sui Prover into Claude Code, and ships five specialized skills — all behind a doc-first agent that reads the docs before writing code. No hallucinated APIs, no deprecated patterns, no stale training data.

Get started View on GitHub Marketplace

The problem

Sui Move evolves rapidly. LLM training data goes stale fast — and agents confidently generate outdated patterns, deprecated APIs, and syntax that no longer compiles.

Before sui-pilot

Stale training data

Claude's knowledge cutoff means it doesn't know about new framework methods, changed APIs, or Move 2024 syntax. It writes code that looks right but doesn't compile.

Before sui-pilot

No real-time feedback

Without LSP integration, Claude can't check whether the code it generates actually type-checks, has the right abilities, or matches the current framework signatures.

With sui-pilot

Doc-first, LSP-verified

Current docs are always in context. The Move LSP catches errors in real time. Every answer is grounded in what works today, not what worked six months ago.

How it works

A slim always-loaded preamble tells the agent where the docs are. Skills invoke procedural workflows. MCP servers provide real-time tooling. No hooks, no matchers, no precomputed indexes — the agent navigates the corpora directly.

Quick start

sui-pilot is a Claude Code plugin. Install it from the marketplace and start asking Sui/Move questions.

1

Install the plugin

From any Claude Code session:

/plugin marketplace add contract-hero/plugin-marketplace
/plugin install sui-pilot@contract-hero
2

Install the Sui toolchain

The Move LSP requires move-analyzer; formal verification requires sui-prover.

curl -sSfL https://raw.githubusercontent.com/MystenLabs/suiup/main/install.sh | sh
suiup install sui
suiup install move-analyzer
3

Restart Claude Code

MCP servers launch at session start. Close and reopen Claude Code to activate the plugin.

4

Try it

Ask a question, run a skill, or get diagnostics:

What are shared objects in Sui and when should I use them?
/move-code-quality
/move-code-review
